# 🎮 Rental PS Manager (Mobile Native)

Aplikasi Mobile Native untuk manajemen operasional Rental PlayStation. Dibangun menggunakan **Kotlin** dan **SQLite**, aplikasi ini mencatat penyewaan, mengotomatisasi perhitungan waktu selesai, menghitung estimasi pendapatan harian, dan terintegrasi langsung dengan WhatsApp pelanggan.

Proyek ini dikembangkan sebagai tugas Ujian Akhir Semester (UAS) mata kuliah Pemrograman Mobile Native, namun dirancang dengan standar logika bisnis yang siap digunakan di dunia nyata.
